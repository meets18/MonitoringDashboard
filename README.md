# Netdata Monitoring Dashboard (DevOps Project)

This project demonstrates automated deployment and testing of a real-time system monitoring dashboard using **Netdata**.

The project simulates a basic DevOps monitoring pipeline where system metrics are collected, visualized, and monitored with automated alerting.

---

## Features

- Automated Netdata installation
- Real-time monitoring dashboard
- CPU / Memory / Disk I/O monitoring
- Custom alert configuration
- Load simulation testing
- Cleanup automation

---

## Tech Stack

- Linux
- Bash scripting
- Netdata
- System monitoring
- DevOps automation
